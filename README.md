# Página de Local Turístico

Projeto desenvolvido durante o curso Full Stack da Rocketseat, utilizando apenas HTML e CSS iniciais.

## Sobre o Projeto

Esta página apresenta um destino turístico, construída com foco em praticar:

- Estruturação semântica em HTML  
- Estilização com CSS puro  
- Reprodução fiel de um layout criado no Figma  
- Conceitos básicos de espaçamento, tipografia e hierarquia visual  
- Organização de conteúdo em seções e cartões informativos  

O objetivo principal foi treinar a construção de interfaces estáticas, mantendo fidelidade ao layout e praticando conceitos fundamentais de HTML e CSS.

## Tecnologias Utilizadas

- HTML5  
- CSS3
